Areia Lab – Laravel SEO Solution

SEO management package with Tailwind UI backend, OpenGraph & Twitter support, and Blade helpers.

Laravel 9+ PHP 8.0+ OpenGraph Twitter Cards

Requirements

  • PHP 8.1 or higher
  • Laravel 9, 10, 11, or 12
  • Database supported by Eloquent (MySQL, PostgreSQL, SQLite, SQL Server)
  • Configured public/storage symlink (php artisan storage:link)
  • Node & NPM (optional for customizing Tailwind UI)

Installation

  1. Add local path repo (optional for local dev): 
    "repositories": [
  { "type": "path", "url": "packages/areia-lab/laravel-seo-solution" }
]
  2. Require the package: 
    composer require areia-lab/laravel-seo-solution
  3. Publish config, views & migrations and run migrations: 
    php artisan vendor:publish --tag=seo-config
php artisan vendor:publish --tag=seo-views
php artisan vendor:publish --tag=seo-migrations
php artisan migrate
php artisan storage:link

Admin Panel

Visit /admin/seo (or your configured prefix) to manage SEO records including uploads for OG/Twitter images.

Types

  • global
  • page
  • model

Meta

  • title
  • description
  • keywords
  • canonical

Social

  • OpenGraph tags
  • Twitter cards
  • Image uploads

Usage in Blade

Render SEO meta tags inside your <head>:

@seoGlobal
@seoPage('contact')
@seoModel($post)
@seoAutoPage

Or fluent API:

{!! app('seo')->global()->render() !!}
{!! app('seo')->forPage('contact')->render() !!}
{!! app('seo')->forModel($post)->render() !!}

Examples

Layout Example (All SEO types) 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>@yield('title', 'My Website')</title>

  @seoGlobal
  @seoAutoPage
  @isset($pageKey)
    @seoPage($pageKey)
  @endisset
  @isset($seoModel)
    @seoModel($seoModel)
  @endisset
</head>
<body>
  @yield('content')
</body>
</html>

Page Example 

@extends('layouts.app')

@section('title', 'Contact Us')

@section('content')
  <h2>Contact Us</h2>
  <p>Feel free to reach out!</p>
@endsection

@push('head')
  @seoPage('contact')
@endpush

Model Example 

@extends('layouts.app')

@section('title', $post->title)

@section('content')
  <h2>{{ $post->title }}</h2>
  <p>{{ $post->body }}</p>
@endsection

@push('head')
  @seoModel($post)
@endpush

Auto Page Detection Example 

@extends('layouts.app')

@section('title', 'About Us')

@push('head')
  @seoAutoPage
@endpush

@section('content')
  <h1>About Our Company</h1>
  <p>We build amazing Laravel packages!</p>
@endsection

Combined Global + Page + Model 

<head>
  @seoGlobal
  @seoAutoPage
  @isset($post)
    @seoModel($post)
  @endisset
</head>

Seeder 

php artisan db:seed --class="AreiaLab\LaravelSeoSolution\Database\Seeders\SeoMetaSeeder"

Configuration 

return [
  'route' => [
    'prefix' => env('SEO_PANEL_PREFIX', 'admin/seo'),
    'middleware' => ['web', 'auth']
  ],
  'disk' => env('SEO_DISK', 'public'),
  'defaults' => ['title_suffix' => env('SEO_TITLE_SUFFIX', '')],
  'cache' => env('SEO_CACHE', true),
  'panel' => ['title_prefix' => 'Areia Lab SEO']
];

Run php artisan storage:link to make uploaded images publicly accessible.
