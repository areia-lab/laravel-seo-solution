Areia Lab – Laravel SEO Solution

SEO management package with Tailwind UI backend, OpenGraph & Twitter support, and Blade helpers.

Install

  1. Add local path repo (optional for local dev): 
    "repositories": [
  { "type": "path", "url": "packages/areia-lab/laravel-seo-solution" }
]
  2. Require the package: 
    composer require areia-lab/laravel-seo-solution:*
  3. Publish & migrate: 
    php artisan vendor:publish --tag=seo-solution-config
php artisan vendor:publish --tag=seo-solution-views
php artisan migrate
php artisan storage:link

Admin Panel

Visit /admin/seo (or your configured prefix) to manage SEO records including uploads for OG/Twitter images.

  • Type: global, page, model
  • Meta: title, description, keywords, canonical
  • OpenGraph: title, description, type, image
  • Twitter: title, description, card, image

Render in Blade

Use Blade directives:

@seoGlobal
@seoPage('contact')
@seoModel($post)

Or fluent API:

{!! app('seo')->global()->render() !!}
{!! app('seo')->forPage('contact')->render() !!}
{!! app('seo')->forModel($post)->render() !!}

Place the output inside your page <head>.

Seeder

php artisan db:seed --class="AreiaLab\LaravelSeoSolution\Database\Seeders\SeoMetaSeeder"

Configuration

return [
  'route' => [
    'prefix' => 'admin/seo',
    'middleware' => ['web','auth']
  ],
  'disk' => 'public',
  'defaults' => ['title_suffix' => '']
];

Run php artisan storage:link so uploaded images are publicly accessible URLs.
